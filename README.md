#### 创建React工程
1、创建普通工程（比较慢）
```
 npx create-react-app hello-react
```
2、创建普通TS工程（比较慢）
```
npx create-react-app hello-tsx --template typescript
```
3、创建组件工程
```
mkdir hello-components && cd hello-components
iceworks init component
```
